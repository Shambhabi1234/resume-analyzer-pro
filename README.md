# Resume Analyzer Pro

A simple web-based resume analyzer that compares a resume with a job description
and calculates a skill match percentage.

## Features
- Resume and job description comparison
- Skill match score visualization
- Highlighted matched and missing skills
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in any web browser

## Future Improvements
- Resume improvement suggestions
- Skill weighting
- PDF resume parsing
